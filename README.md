# Kakao-P2-NoteApp
(안내) 공부를 위한 프로젝트로 .env 파일은 .gitignore에 포함하지 않았습니다.

<h2>실행 안내</h2>

1. 설치
```
npm install
```

2. 빌드
```
npm run build
```

3. 로컬 서버 구동

|환경|경로|
|------|---|
|sandbox|npm run sandbox|
|cbt|npm run cbt|
<br>
